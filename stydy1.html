関数の作り方

  1. 複数の処理をまとめて実行出来るもの
  2. 関数は、「function 変数名(){ }」を利用
  3. 戻り値の指定が欲しい場合には、return を利用

ブラウザを操作してみよう

  1. JavaScriptとは、「ブラウザ内のデータを動的に扱う言語」

  2. Browser自体が、JavaScriptのオブジェクト : BOM(Browser Object Model)

  3. ブラウザのウィンドウ(window)自体にアクセスが出来る 

      <script>
        window.location.href = "http://www.google.com/"
      </script>

ブラウザ内の文章を操作してみよう

  1. 大体の人が利用する分野が、このへん

  2. ブラウザ内の文章(Document)オブジェクト : DOM (Document Object Model)

  3. 色を変えたり、レイアウトを変えたり。

      <script>
        document.body.bgColor='red';
      </script>

JavaScriptといえば、「DOMの操作」と認識される方が多い
